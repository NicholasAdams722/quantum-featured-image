# Quantum Featured Image Generator

A single-file HTML tool for generating branded featured images for Quantum Letter posts. Renders to a canvas and downloads as PNG.

## Usage

Open `index.html` in any modern browser. No build step, no dependencies.

```bash
open index.html
```

## Features

- **Post types** — Quantum Letter, Quantum Update, Investor Update, Weekend Investor Update
- **Themes** — Blue and Green gradient presets
- **Backgrounds** — Smooth gradient or textured, with optional uploaded background image and adjustable wash
- **Sizes** — Feature (1920×1080) or Social (1200×630)
- **Output** — Full-resolution PNG download with auto-generated filename based on title and date

The brand frame, logo, and gradient are fixed — only text and background imagery change.
